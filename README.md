# Cálculo de Viáticos – Viaje CDMX ↔ Xalapa

Este repositorio contiene el cálculo aproximado de los viáticos del viaje con dos trayectos principales:

1. **Aéreo Juárez → CDMX** (solo para quienes vienen de Juárez)  
2. **Camión CDMX → Xalapa** (para todas las personas, con descuento aplicado)

Además se considera el hospedaje dividido entre cinco personas.

## Tarifas utilizadas
