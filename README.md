# Cálculo de Viáticos – Viaje CDMX ↔ Xalapa

Este repositorio contiene el cálculo aproximado de los viáticos del viaje con dos trayectos principales:

1. **Aéreo Juárez → CDMX** (solo para quienes vienen de Juárez)  
2. **Camión CDMX → Xalapa** (para todas las personas, con descuento aplicado)

Además se considera el hospedaje dividido entre cinco personas.

---

## Tarifas utilizadas

| Concepto | Costo |
|----------|-------|
| 1. Transporte local | $700 |
| 2. Traslado adicional | $800 |
| 3. Otro traslado | $800 |
| 4. Boleto de avión | $1,600 |
| 5. Hospedaje (total / 5 personas) | 4,985 / 5 = **$997** |
| 6. Descuento aplicado al camión | ✔️ |
| 7. Porcentaje del descuento | 50% |

## Cálculo del camión CDMX → Xalapa

- Tarifa base del camión: **$800**  
- Descuento aplicado: **50%**  
- Costo final por persona:  
  **$800 × 0.5 = $400**

---

## Cálculo del avión Juárez → CDMX

- Costo avión (solo ida, para quienes vienen de Juárez): **$1,600**

Las personas de CDMX **no pagan este tramo**, pero **sí pagan el regreso** si vuelven a Juárez.
